# Lumie

Lumie is a voice-responsive AI assistant that operates in real-time, utilizing OpenAI’s GPT-4o Realtime API. In contrast to typical voice assistants, Lumie features a visible RGB LED body that responds to her "mood" and speech patterns via an Arduino Uno R4.

——————————————————————————————————————————————————————————————————    

**Features:**   

* Voice & Audio logic

* RGB Fading/Gradients

* Ultrasonic Sensing (In Progress)
